### Self-Host - APIs for user management.

Intergate Hoppscotch self-host with your organization's user management process with our new APIs. You can now perform user management on Hoppscotch self-host and perform actions such as inviting users and granting admin rights using APIs.

### Generate JUnit Reports

JUnit reports are essential for documenting test results, integrating with CI tools, You can now generate a JUnit Report from your collection run via the Hoppscotch CLI!
