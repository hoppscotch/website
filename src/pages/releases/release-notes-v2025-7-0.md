#### Improved Self-Hosted Onboarding Flow

The self-hosted onboarding flow has been revamped to provide a smoother and more intuitive setup experience. This update simplifies the initial configuration process, making it easier for users to get started with their self-hosted instance.

#### Enhanced Authorization Panel UI

The authorization panel UI has been improved for better usability and clarity. This update enhances the overall user experience when managing authentication methods, making it easier to configure and use various authorization options.

#### Experimental Scripting Editor

We've introduced an experimental scripting editor that allows users to create and manage scripts within Hoppscotch. This feature aims to provide advanced users with more flexibility and control over their API requests and workflows.
