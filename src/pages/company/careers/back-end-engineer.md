# Back-end Engineer

<span class="inline-flex px-4 py-1 rounded bg-accent text-accentContrast">Location: Remote</span>

### Overview

We are looking for an experienced backend engineer to join our dynamic team. You will be responsible for maintaining and managing the back-end infrastructure of our software applications.

If you have the ability to develop high-level applications as per the requirements, please do apply for this job.

### Summary

As a back-end engineer, you will design and develop backend applications i.e. the database server, server-side applications as well as back-end system architecture. You will be working on the web as well as mobile applications. You should have expertise in REST, GraphQL API development, and database management.

### Responsibilities

- Be an active contributor during the project specification and requirement gathering phase
- Use application logic to check the feasibility of the application
- Planning and implementing best practices and new database designs
- Write reusable code blocks and finish the coding within the stipulated time
- Test out your code before sending it forward to the Testing Team
- Follow Agile development methodologies
- Create and integrate the database with its complete maintenance
- Design frameworks to develop server-side software
- Identify problem areas and bring relevant changes
- Collaborate with front end developer for integration of the application
- Perform UI tests to ensure maximum performance
- Work with front-end developers and team members for better evaluation and design codes and programs accordingly
- Follow the latest technological trends and emerging technologies

### Qualifications

- Proven 2+ years of experience as a back-end engineer or full-stack engineer in any enterprise developer tools
- Prior experience with cloud platforms like Google Cloud Platform is nice to have
- Possess amazing knowledge of CMS framework
- Fair knowledge of web server technologies and server-side programming
- Exceptional knowledge of various programming languages like JavaScript, TypeScript etc
- Good experience of working with building back-end APIs
- Strong understanding of web development, programming tools, and frameworks like Node.js, Nest.js, PostgreSQL, etc
- Knowledge of database management systems
- Proficient in written and verbal communication
- Outstanding problem-solving skills
- Should be a team player
- The ability to multitask is a must
- Attention to detail
