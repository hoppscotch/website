Hoppscotch v2024.12.0: Generate scripts with AI, Sync Admin role from SAML and more

Kiran Johns

Hoppscotch builds an easy-to-use, developer-friendly, powerful API testing client. Our v2024.12.0 release brings exciting updates to enhance your experience with Hoppscotch.

Try it out—start using our API web client easily! We constantly seek improvement, so share your feedback and suggestions at hello@hoppscotch.io.

Generate pre-request and post-request scripts using prompts

Hoppscotch's intuitive prompt-based system helps developers of all skill levels generate custom pre-request and post-request scripts effortlessly. This makes it accessible and straightforward for anyone to implement powerful request automation.

Bulk edit on request body form-data

The new bulk editing capability lets you modify multiple form fields easily and manage large datasets efficiently.

Sync Admin role from SAML

We now synchronize administrative privileges between your SAML provider and Hoppscotch self-host enterprise, which enables automatic admin role assignment and maintains consistent access management across your organization.

Bug fixes and improvements

We have fixed minor issues and improved performance to ensure a smoother, more reliable testing experience for our users.
